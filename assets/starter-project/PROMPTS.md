# 5 готовых промптов — копируй и вставляй

> Выдели текст промпта целиком, скопируй и вставь в чат Claude Code.
> Не бойся менять детали под себя — это шаблоны, а не заклинания.

---

## 1. Сделать лендинг (страницу-визитку)

```
Сделай одностраничный сайт-визитку в одном файле landing.html.
Тема: [впиши свою — например, «домашняя кондитерская» или «репетитор по математике»].
Нужно: шапка с названием, 3 преимущества с иконками-эмодзи, блок «обо мне»,
кнопка «Написать в Telegram» (ссылку поставь-заглушку), приятные цвета, всё по-русски.
Стили — прямо внутри файла, без внешних библиотек. Когда закончишь — открой файл в браузере.
```

---

## 2. Разобрать таблицу CSV

```
В этой папке лежит файл example.csv — таблица продаж.
Изучи его и расскажи простыми словами: какой товар принёс больше всего денег,
какой город лидирует по продажам, есть ли что-то странное в данных.
Потом сделай файл report.html с 2–3 наглядными диаграммами по этим данным
(всё в одном файле, без интернета) и открой его в браузере.
```

---

## 3. Написать Telegram-бота

```
Напиши простого Telegram-бота на Python в файле bot.py.
Что он умеет: на /start здоровается и рассказывает, что умеет;
на любое текстовое сообщение отвечает [впиши идею — например, «случайной цитатой
про успех» или «переводит текст в ЗАБОРЧИК иЗ бОлЬшИх И мАлЕнЬкИх БуКв»].
Токен бота бери из переменной окружения BOT_TOKEN (объясни мне, как получить токен
у @BotFather и как запустить бота на моём компьютере — по шагам, для новичка).
```

---

## 4. Навести порядок в папке с файлами

```
Напиши скрипт organize.py, который наводит порядок в папке:
раскладывает файлы по подпапкам по типу (Картинки, Документы, Видео, Архивы, Прочее).
Сначала он должен показать план «что куда переедет» и спросить подтверждение,
и только потом перемещать. Ничего не удалять. Покажи мне, как запустить его
на моей папке Загрузки, но сначала прогони в тестовом режиме.
```

---

## 5. Сделать презентацию

```
Сделай презентацию на 5 слайдов в одном файле slides.html
(слайды листаются стрелками на клавиатуре).
Тема: [впиши свою — например, «почему нашей семье нужен отпуск на море»].
Структура: титульный слайд, проблема, решение, выгоды, финальный призыв.
Оформи красиво: крупные заголовки, эмодзи, контрастные цвета. Всё по-русски.
Когда закончишь — открой в браузере.
```

---

## Подсказки

- **Продолжай диалог.** После ответа можно писать «сделай кнопку побольше», «добавь тёмную тему», «объясни, что делает эта строка» — Claude помнит контекст.
- **Просите объяснений.** «Объясни как для новичка» — работает.
- **Что-то пошло не так?** Так и напиши: «не работает, вот что я вижу: …» — и вставь текст ошибки.
