# Мой первый проект с Claude Code

> Этот файл Claude читает автоматически при каждом запуске в этой папке.
> Здесь — правила, как ему со мной работать. Можешь менять их под себя.

## Кто я

- Я новичок: объясняй просто, без жаргона. Если термин нужен — сразу расшифруй в скобках.
- Общайся со мной по-русски. Код и комментарии в коде — на английском (так принято).

## Как работать

1. **Маленькие шаги.** Делай задачу по частям и показывай результат после каждой части.
2. **Сначала скажи, что собираешься сделать** — одной-двумя фразами, потом делай.
3. **Ничего не удаляй без спроса.** Перед удалением файлов или больших кусков кода — спроси меня.
4. **Если что-то сломалось** — не паникуй и не переписывай всё заново. Найди причину, объясни её мне простыми словами и предложи минимальное исправление.
5. **Файлы создавай в этой папке** (или в подпапках), а не где-то в системе.
6. **Готовый результат покажи** — открой страницу в браузере, запусти скрипт, выведи таблицу. Мне важно увидеть, что оно работает.

## Что лежит в этой папке

| Файл | Зачем |
|------|-------|
| `CLAUDE.md` | Этот файл — правила для Claude |
| `PROMPTS.md` | 5 готовых промптов — открой и копируй |
| `example.csv` | Маленькая таблица с данными — для тренировки анализа |
| `.gitignore` | Список того, что не нужно сохранять в git |

## С чего начать

Открой `PROMPTS.md`, скопируй любой промпт целиком и вставь в чат Claude.
Первый запуск лучше начать с промпта №1 (лендинг) — результат видно сразу в браузере.
