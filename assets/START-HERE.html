Шаг 1 — открой Claude Code

✳ В панели VS Code проще

Открой VS Code (найди в меню «Пуск»; на Mac — в Applications; или Cursor, если выбрал его) и нажми Ctrl+Esc — панель Claude откроется сразу. Или нажми значок ✳: он в правом верхнем углу над кодом, в левой панели или внизу окна.

Значок ✳ — справа вверху над кодом (появляется, когда открыт файл). Источник: официальная документация Claude Code.

Значка нет? Чаще всего потому, что не открыт ни один файл — открой любой, и значок появится. Не помогло: Ctrl+Shift+P → Claude Code: Open in New Tab. Подробно — в ФАК ниже.

Не перепутай с другими чатами. У Cursor (если выбрал его) есть свой ИИ-чат (Ctrl+L) — это другой продукт с отдельной оплатой. Пиши только в панель со значком ✳.

⌨ В терминале

В VS Code: Terminal → New Terminal (или Ctrl+`), набери claude и Enter. Работает в любом терминале системы — команду поставил установщик. Пишет «claude не найдена»? Закрой терминал и открой заново — команда подхватится.

Три кнопки безопасности
⎋

Esc — стоп

Claude делает не то? Нажми Esc — он остановится сразу. Писать в чат «стой, не надо» бесполезно: сообщение просто встанет в очередь, а Claude тем временем продолжит. То, что он успел сделать, сохранится.

↺

/rewind — откатить назад

Claude испортил файл? В панели: наведи курсор на любое сообщение в чате — рядом появится кнопка отката (Rewind), выбери «Rewind code». В терминале: набери /rewind или дважды нажми Esc при пустом поле ввода. Вернуть можно код, диалог или и то и другое — такую точку Claude создаёт перед каждой своей правкой, сам.

Важное ограничение: откат возвращает только те файлы, которые Claude правил напрямую. То, что натворили команды в терминале (удаление, перемещение), он вернуть не сможет.

В интернете часто советуют запускать Claude с ключом --dangerously-skip-permissions, «чтобы не спрашивал каждый раз». Не делай этого. Эти вопросы — единственное, что стоит между Claude и твоими файлами.
Лимиты — чтобы не удивляться
%

Квота одна на всё

Подписка даёт не безлимит, а квоту — и она общая: чат на claude.ai, приложение Claude и Claude Code черпают из одного котла. Поэтому «я сегодня почти не кодил, а лимит кончился» обычно означает, что утром ты переписывался в браузере.

Посмотреть, сколько осталось и когда обновится: набери /usage.

⏳

«Limit reached» — это не поломка

Если Claude написал, что лимит исчерпан, — ничего не сломалось и ничего не потерялось. Время сброса указано прямо в сообщении, дождись его. Лимитов два: на короткое окно в несколько часов и на неделю.

Растянуть квоту просто: не держи включённой самую тяжёлую модель без надобности (переключить — /model), а новую задачу начинай с /clear.

Случайно начать платить сверх подписки нельзя
Доплата за перерасход по умолчанию выключена — её надо включать вручную и привязывать карту. Если счета «за API» всё-таки приходят при живой подписке, набери /status и проверь, каким аккаунтом ты вошёл (разбор — в ФАК).
Если ты в России
🌐

Почему браузер открывается, а Claude Code — нет

Claude и другие ИИ ограничивают доступ из России со своей стороны — по адресу, с которого приходит запрос. Отсюда самая частая ловушка: claude.ai в браузере открывается, а Claude Code выдаёт ошибку. Так бывает, когда зарубежный адрес есть только у браузера (например, из-за расширения), а Claude Code — отдельная программа: её запросы идут напрямую, с российского адреса.

Для работы включи AI-мост из установщика — он даёт стабильный зарубежный адрес только ИИ-сервисам (Claude и другим), а весь остальной интернет идёт как обычно. Если не выбирал его при установке — запусти установщик ещё раз и отметь пункт «AI-мост». Если пользуешься своим VPN — подойдёт и он, при условии что он покрывает весь компьютер, а не только браузер.

🔒

Адрес должен быть стабильным всю сессию

Адрес проверяется на каждом запросе, а не только при входе. Пропал зарубежный адрес на минуту — Claude перестанет отвечать или выбросит из аккаунта. AI-мост для того и сделан: он держит один и тот же адрес всё время работы. Если пользуешься своим VPN — не отключай его посреди сессии и не прыгай между странами: выбери одну и держись её.

💳

Оплата — и о рисках честно

Российские карты не проходят. Работают: карта банка страны, которую Anthropic обслуживает (Казахстан, Армения, Грузия, Турция, Сербия, ОАЭ), либо оплата внутри приложения Claude через зарубежный Apple ID: месячный Pro там стоит те же $20, дороже выходят годовой и Max-планы (комиссия Apple). Для зарубежного Apple ID всё равно нужен способ оплаты его страны — например, подарочная карта того же региона.

Риск, о котором лучше знать заранее. В 2026-м сообщали о волне блокировок аккаунтов у пользователей из России, а апелляции удовлетворяют редко. Чаще всего среди причин называют: оплату «общей» картой посредника, скачущий IP из дешёвых VPN-пулов, свежесозданную почту и виртуальный SMS-номер. Отсюда простые выводы: одна страна VPN, давно живущий почтовый ящик — и не храни в чатах единственную копию чего-то важного.

Частые проблемы — ФАК (нажми, чтобы раскрыть)
Редактор (VS Code или Cursor) открыт, а Claude нигде нет — панели и значка ✳ не видно
Самая частая ситуация. Открой панель вручную — любым способом:
1. Нажми Ctrl+Shift+P (палитра команд) → набери Claude Code: Open in New Tab → Enter. Самый надёжный способ.
2. Или открой Extensions (значок кубиков ▦ слева, или Ctrl+Shift+X) → в списке Installed («Установленные») найди «Claude Code for VS Code» (издатель Anthropic, оранжевый ✳) → открой его → в правом верхнем углу появятся ⋯ и значок Claude → выбери «Claude Code: Open in New Tab».
3. Или нажми «✱ Claude Code» в строке состояния — в правом нижнем углу окна. Работает, даже если ни один файл не открыт.
Если значка ✳ нет в тулбаре сверху: открой любой файл (без открытого файла он скрыт); проверь версию редактора (нужна свежая); перезапусти (палитра → Developer: Reload Window); временно отключи другие ИИ-расширения (Cline, Continue, Codex); убедись, что окно не в режиме «Restricted Mode» — он включается, если на вопросе о доверии к папке нажать «No»; переоткрой папку и ответь «Yes, I trust the authors».
Выбрал Cursor, и он просит «Upgrade to Pro» — мне надо платить за Cursor?
Нет. Cursor и Claude — разные подписки. Cursor спокойно оставляй на бесплатном Free Plan — это ничему не мешает. Claude Code работает по твоей подписке Claude (Pro/Max), которую ты подключил при входе. Апгрейд Cursor покупать не нужно.
Расширения Claude Code вообще нет в списке
Открой Extensions (Ctrl+Shift+X) → в поиске набери Claude Code → выбери официальное от Anthropic (не похожее с другим названием) → Install. Установщик ставит его сам, но так вернёшь, если пропало.
В терминале пишет «claude не найдена»
Закрой терминал и открой заново (или перезапусти VS Code) — обновится PATH, и команда подхватится.
Пишет «Not logged in · Please run /login»
Набери в панели или терминале /login и войди в открывшемся браузере — тем аккаунтом, где оформлена подписка.
Claude не отвечает или ошибка сети
Из России Claude ограничивает доступ со своей стороны — включи AI-мост из установщика (см. раздел «Если ты в России» выше); свой VPN на весь компьютер тоже подойдёт. Проверь интернет. Запусти claude в терминале — там ошибка видна подробнее.
Claude только рассказывает, что сделает, но файлы не меняет
Скорее всего, ты случайно включил режим планирования — клавиша Shift+Tab живёт по соседству с Shift+Enter, её легко задеть.
Посмотри на строку под полем ввода: там написан текущий режим. Нажимай Shift+Tab, пока не вернёшься в обычный режим, — или просто прими предложенный план.
Claude завис — «думает» уже пятнадцать минут
По порядку, до первого сработавшего:
1. Отправь любое короткое сообщение — подвисший ответ часто оживает.
2. Нажми Esc.
3. Если ты в России — проверь, что AI-мост включён (значок в трее), или переподключи свой VPN. Оборванное соединение даёт ровно такой «вечный спиннер».
4. Закрой панель или терминал и открой заново. Диалог не потеряется — набери claude --resume в той же папке, и разговор вернётся.
5. Набери /doctor — он сам проверит установку и предложит починить.
У меня подписка, а Claude просит API-ключ или приходят счета «за API»
Набери /status — он покажет, каким способом ты вошёл. Причин ровно две.
1. В системе прописана переменная ANTHROPIC_API_KEY — она перебивает подписку. Удали её из переменных окружения и перезапусти редактор.
2. Ты вошёл аккаунтом Anthropic Console (это оплата по факту, за каждый запрос) вместо аккаунта с подпиской. Набери /login и выбери Claude account.
Лимит кончается слишком быстро
Квота общая для чата на claude.ai, приложения Claude и Claude Code — переписка в браузере тратит ту же квоту, что и работа с кодом.
Сильнее всего квоту съедает самая тяжёлая модель. Держи по умолчанию обычную, а тяжёлую включай точечно — командой /model.
Проверить остаток и время сброса: /usage. Длинные разговоры тоже дороже коротких — начинай новую задачу через /clear.
Claude забыл, о чём мы говорили, и начал всё ломать
У него ограниченная память на один разговор. Когда она заполняется, Claude сжимает диалог — и часть подробностей теряется.
Лечится привычкой: одна задача — один разговор. Закончил задачу — /clear, и начинай следующую с чистого листа.
Для долгого проекта попроси Claude вести файл NOTES.md с текущим состоянием дел. Тогда новый разговор можно начать словами «прочитай NOTES.md и продолжай».
Русский текст превращается в кракозябры (Windows)
Классическая беда Windows-терминала с кодировкой. Перед запуском Claude выполни в терминале команду chcp 65001 — она включает UTF-8.
Если повторяется постоянно — поставь PowerShell 7 (в старом встроенном терминале это лечится хуже) и попроси Claude сохранять файлы в UTF-8.
Скриншот не вставляется — Ctrl+V не работает
На Windows вставка картинки — Alt+V. На Mac — Ctrl+V (именно Ctrl, не Cmd).
Не сработало — сохрани картинку файлом и перетащи её в поле ввода панели, удерживая Shift (без Shift редактор просто откроет файл). В терминале — обычное перетаскивание: вставится путь к файлу.
Claude испортил файл — как вернуть, как было?
1. В панели наведи курсор на сообщение в чате и нажми кнопку отката (Rewind). В терминале — /rewind (или дважды Esc при пустом поле). Выбери точку до правки.
2. Не помогло — правый клик по файлу в VS Code → Open Timeline → выбери прошлую версию. Редактор хранит их сам, независимо от Claude.
Помни ограничение: /rewind не отменяет то, что сделали команды в терминале (например, удаление файла).
А он не сломает мне компьютер?
Именно поэтому Claude спрашивает разрешение перед каждым действием с файлами и показывает, что собирается сделать. Читай и жми Yes — или No, если не понял.
Работает он в той папке, которую ты открыл, а не «по всему компьютеру».
Единственный реальный способ обжечься — самому отключить эти вопросы ключом --dangerously-skip-permissions, который советуют в интернете. Не отключай.
Закрыл ноутбук — а задание не доделалось
Claude Code работает на твоём компьютере, а не где-то в облаке. Компьютер уснул, крышка закрылась, терминал закрыли — работа остановилась на полпути.
Ничего страшного: открой заново и напиши продолжай. На долгих задачах просто не давай машине уснуть.
Это бесплатно? И чем оно отличается от ChatGPT?
Нет, не бесплатно. Claude Code работает только с платной подпиской Claude — на бесплатном аккаунте он не запустится.
Отличие от чата. ChatGPT и чат Claude отвечают тебе словами: ты копируешь код руками. Claude Code сам создаёт и правит файлы у тебя на компьютере, запускает команды и проверяет результат.
В интернете советуют npm install, WSL или Linux внутри Windows
Это устаревшие инструкции. Установщик уже поставил всё, что нужно, правильным современным способом.
Ничего не доустанавливай по чужим гайдам — так проще всего сломать рабочую установку. Если что-то не работает, начни с /doctor и с других пунктов этого раздела.
Ничего не помогло

Продвинутое: доп. сервисы, API-ключ, голосовой ввод
Дополнительные ИИ-сервисы (картинки, видео, озвучка, поиск) работают по API-ключам. Впиши их в мастер-экране установщика («Сохранить ключи») или в файл .claude/.credentials.master.env в домашней папке — по одному КЛЮЧ=значение на строку. Всё необязательно.
Аккаунт Console вместо подписки. На шаге входа выбери «Anthropic Console» и войди в браузере аккаунтом Console — это оплата по факту использования, не подписка. Если нужен именно API-ключ — пропиши его в переменную окружения ANTHROPIC_API_KEY: Claude спросит разовое подтверждение при запуске.
Голосовой ввод. Установи Handy — в любом поле зажми Ctrl+Shift+Z, говори, отпусти: текст вставится сам. Описать задачу голосом — 10 секунд вместо 2 минут печати.
Твои правила для Claude. Файл CLAUDE.md в папке проекта Claude читает автоматически — допиши «отвечай кратко», «показывай результат в браузере», и он будет слушаться в каждом диалоге.
Что теперь стоит на компьютере
VS Code — редактор кода; панель Claude работает внутри него (Cursor — опциональная альтернатива, если выбрал его).
Claude Code — сам ИИ-агент: и команда claude в терминале, и панель-расширение.
Git — хранит версии твоих проектов (Claude пользуется им сам).
Стартовый проект HamidunStart — с правилами и промптами.
