Первые 10 минут с Claude Code

Установка закончена. Этот чеклист работает без интернета — пройди шаги сверху вниз, и через 10 минут у тебя будет первый результат.

1

Открой Cursor ~1 мин

Ярлык Cursor появился на рабочем столе / в меню приложений. При первом запуске он спросит пару настроек — соглашайся с вариантами по умолчанию.

2

Открой стартовый проект ~1 мин

В Cursor: File → Open Folder и выбери папку Hamidun-Start — она лежит в твоей домашней папке (на Windows это C:\Users\<имя>, на Mac — /Users/<имя>). Внутри уже есть правила для Claude и готовые промпты.

3

Запусти Claude Code ~2 мин

  • Открой встроенный терминал: меню Terminal → New Terminal (или Ctrl+`).
  • Набери команду claude и нажми Enter.
  • Или нажми на иконку Claude на боковой панели — расширение уже установлено.
4

Войди в аккаунт ~2 мин

При первом запуске Claude предложит войти — откроется браузер. Войди в свой аккаунт Claude (подписка Pro/Max) и вернись в Cursor. Это нужно один раз.

5

Первый промпт ~3 мин

Открой файл PROMPTS.md в стартовом проекте, скопируй промпт №1 («Сделать лендинг»), вставь в чат Claude и нажми Enter. Наблюдай: Claude создаст файл и сам откроет его в браузере.

6

Продолжи диалог ~1 мин

Попроси что-нибудь поменять своими словами: «сделай фон темнее», «добавь ещё один блок», «объясни, что ты сделал». Claude помнит контекст разговора.

Совет: файл CLAUDE.md в папке проекта — это правила, которые Claude читает автоматически. Допиши туда своё: «отвечай кратко», «всегда показывай результат в браузере» — и он будет слушаться.
Что-то не работает?
  • Команда claude не найдена — закрой и открой терминал заново (или перезапусти Cursor).
  • Claude не отвечает или ругается на сеть — проверь интернет и VPN/AI-мост, затем повтори.
  • Остальное — напиши в бот поддержки: t.me/HamidunAcademyBot.
