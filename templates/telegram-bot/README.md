# Шаблон: Telegram-бот

Минимальный рабочий бот на Python (библиотека aiogram).
Умеет отвечать на /start, /help и повторять любые сообщения (эхо).
Это фундамент — дальше наращивайте свои команды.

## Что внутри

| Файл | Что это |
|------|---------|
| `bot.py` | Весь код бота (около 80 строк с комментариями) |
| `requirements.txt` | Список библиотек для установки |
| `.env.example` | Образец файла с настройками (токен бота) |
| `.gitignore` | Защита: не даёт случайно выложить секреты в git |

## Шаг 1. Получите токен бота

1. Откройте Telegram и найдите бота **@BotFather** (официальный, с галочкой).
2. Отправьте ему команду `/newbot`.
3. Придумайте имя бота (любое) и юзернейм (должен заканчиваться на `bot`,
   например `my_first_test_bot`).
4. BotFather пришлёт токен — длинную строку вида
   `<цифры>:<буквы_и_цифры>` (сначала число, потом двоеточие и ~35 символов).
   Скопируйте её. Никому не показывайте: токен = полный доступ к боту.

## Шаг 2. Настройте проект

1. Скопируйте файл `.env.example` и назовите копию `.env`
   (именно так, с точкой в начале).
2. Откройте `.env` и вставьте токен после знака `=`:

   ```
   BOT_TOKEN=сюда_вставьте_ваш_токен
   ```

## Шаг 3. Установите зависимости

Нужен Python 3.10 или новее. В терминале, находясь в этой папке:

```
pip install -r requirements.txt
```

## Шаг 4. Запустите бота

```
python bot.py
```

В терминале появится «Бот запущен». Откройте своего бота в Telegram,
нажмите «Старт» — он ответит. Остановить бота: `Ctrl+C` в терминале.

Важно: бот работает, только пока запущен `bot.py` на вашем компьютере.
Закрыли терминал — бот «уснул».

## Как развивать бота с ИИ-помощником

Откройте эту папку в Cursor или запустите Claude Code и просите
простым языком:

- «Добавь команду /about с рассказом о боте»
- «Сделай кнопки под сообщением: Да / Нет»
- «Пусть бот отвечает на фото комплиментом»
- «Добавь команду, которая присылает случайный совет дня»
- «Объясни, как выложить бота на сервер, чтобы он работал круглосуточно»

Совет: после каждого изменения перезапускайте бота
(`Ctrl+C`, затем снова `python bot.py`) — иначе новые команды не подхватятся.
