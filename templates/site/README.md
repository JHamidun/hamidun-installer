# Шаблон: Лендинг (сайт-визитка)

Готовая основа для одностраничного сайта: шапка, главный экран, блок
возможностей, блок «О проекте», форма и подвал. Чистые HTML + CSS + JS —
без сборщиков и зависимостей. Открыл — и работает.

## Что внутри

| Файл | Что это |
|------|---------|
| `index.html` | Структура страницы (текст, блоки) |
| `style.css` | Все стили. Цвета и размеры — в переменных `:root` наверху |
| `script.js` | Немного интерактива: мобильное меню, форма-заглушка |

## Как запустить

Вариант 1 — самый простой:

1. Откройте папку в проводнике.
2. Дважды кликните на `index.html` — сайт откроется в браузере.

Вариант 2 — через локальный сервер (страница будет обновляться правильнее):

1. Откройте папку в Cursor или VS Code.
2. В терминале выполните:

   ```
   npx serve .
   ```

3. Откройте в браузере адрес, который покажет команда (обычно `http://localhost:3000`).

## Как работать с ИИ-помощником (Claude Code / Cursor)

Откройте эту папку в редакторе и пишите запросы простым языком.
Примеры промптов, с которых можно начать:

- «Поменяй цветовую схему на светлую с зелёным акцентом»
- «Добавь секцию с тарифами: три карточки с ценами»
- «Добавь блок с отзывами клиентов после блока возможностей»
- «Сделай, чтобы форма отправляла данные мне в Telegram» (помощник сам
  спросит, что для этого нужно)
- «Замени тексты-заглушки: мой проект — это [опишите свой проект]»

Совет: просите по одному изменению за раз и смотрите результат в браузере
(обновите страницу клавишей F5). Так проще понять, что именно поменялось.

## Куда двигаться дальше

- Замените все тексты-заглушки на свои.
- Подберите свои цвета в `style.css` (блок `:root`).
- Добавьте картинки: положите их в папку `images/` рядом и подключите
  через `<img src="images/имя-файла.png">`.
- Когда сайт готов — попросите ИИ-помощника: «помоги опубликовать этот
  сайт бесплатно», он предложит варианты.
