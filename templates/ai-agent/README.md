# Шаблон: AI-агент (консольный чат с моделью)

Минимальный агент на Python: вы задаёте вопросы в терминале, языковая
модель отвечает и помнит контекст беседы. Работает с любым провайдером,
у которого OpenAI-совместимый API (таких большинство).

## Что внутри

| Файл | Что это |
|------|---------|
| `agent.py` | Весь код агента (цикл «вопрос → ответ» с историей) |
| `requirements.txt` | Список библиотек для установки |
| `.env.example` | Образец файла настроек (ключ, адрес API, модель) |
| `.gitignore` | Защита: не даёт случайно выложить секреты в git |

## Шаг 1. Получите API-ключ

Нужен ключ любого провайдера языковых моделей. Варианты:

- **OpenAI** — зарегистрируйтесь на platform.openai.com, раздел API Keys.
- **Другой провайдер** (совместимый с OpenAI API) — возьмите ключ и
  base URL из его документации. Подойдут многие сервисы, в том числе
  российские и локальные (например, Ollama на вашем компьютере).

Ключ — это секрет. Не публикуйте его и не вставляйте в код напрямую.

## Шаг 2. Настройте проект

1. Скопируйте файл `.env.example` и назовите копию `.env`.
2. Откройте `.env` и заполните:

   ```
   LLM_API_KEY=сюда_вставьте_ваш_ключ
   LLM_BASE_URL=
   LLM_MODEL=
   ```

   - `LLM_BASE_URL` оставьте пустым, если используете официальный OpenAI.
     Для другого провайдера впишите его адрес API, например
     `https://api.example.com/v1` (точный адрес — в документации провайдера).
   - `LLM_MODEL` тоже можно оставить пустым — возьмётся `gpt-4o-mini`.
     Или впишите название модели вашего провайдера.

## Шаг 3. Установите зависимости

Нужен Python 3.10 или новее. В терминале, находясь в этой папке:

```
pip install -r requirements.txt
```

## Шаг 4. Запустите агента

```
python agent.py
```

Пишите вопросы и жмите Enter. Агент помнит предыдущие реплики, так что
можно вести диалог. Выход — пустая строка или `Ctrl+C`.

## Как развивать агента с ИИ-помощником

Откройте эту папку в Cursor или запустите Claude Code и просите
простым языком:

- «Поменяй характер агента: пусть он будет строгим учителем математики»
  (характер задаётся переменной `SYSTEM_PROMPT` в `agent.py`)
- «Добавь команду /save, которая сохраняет весь диалог в текстовый файл»
- «Научи агента читать текстовый файл и отвечать на вопросы по нему»
- «Сделай, чтобы агент умел искать в интернете»
- «Преврати консольный чат в Telegram-бота» (пригодится соседний
  шаблон `telegram-bot`)

Совет: начинайте с маленьких изменений и запускайте агента после каждого,
чтобы сразу видеть результат.
