# Стартовые шаблоны: с чего начать свой проект

Три готовые заготовки для первых проектов. Выберите ту, что ближе к вашей
идее, откройте папку в Cursor (или запустите в ней Claude Code) — и
развивайте проект, общаясь с ИИ-помощником простым языком.

| Папка | Что это | Технологии |
|-------|---------|------------|
| `site/` | Одностраничный сайт (лендинг) | HTML + CSS + JavaScript |
| `telegram-bot/` | Telegram-бот с командами и эхо-ответом | Python + aiogram |
| `ai-agent/` | Консольный чат с языковой моделью | Python + OpenAI-совместимый API |

## Как пользоваться

1. **Скопируйте** нужную папку в отдельное место (например, `Документы/мой-проект`) —
   оригинал останется чистым для следующих экспериментов.
2. Откройте копию в редакторе и прочитайте `README.md` внутри —
   там пошаговая инструкция запуска.
3. Запустите шаблон по инструкции и убедитесь, что он работает.
4. Дальше — просите ИИ-помощника вносить изменения. Примеры промптов
   есть в каждом README.

## Три правила безопасности

1. Файл `.env` с вашими ключами и токенами — секретный. Не отправляйте
   его никому и не выкладывайте в интернет.
2. В шаблонах вместо `.env` лежит образец `.env.example` — скопируйте его
   в `.env` и заполните своими значениями.
3. Если случайно засветили ключ — просто создайте новый у провайдера,
   а старый удалите.
